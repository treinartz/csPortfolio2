Week3: Loops

Draw:Here are some small examples of how to draw things. Read and experiment with the code. Then, either try to reproduce the code or try your own by "spinning off," or "remixing" what you see here. The essence of learning is through exploration and mistakes cast as learning opportunities.

Modular division: good for cycles



var x = 0;
function setup() {
  createCanvas(450, 450);
}
function draw() {
  background(220);
  translate(0,height/4);
  noStroke();
  fill(255,0,0);
  rect(x,0, 200,200);
  x++;
  x = x % width;
}

Elastic ball (using sin())


function setup(){
  createCanvas(400,400);
}
function draw(){
  background(255);
  var amplitude = height/2;//play with the amplitude value 
  var y = sin(radians(frameCount))* amplitude;
  translate(width/2,height/2);
  ellipse(0,y,50,50);
}
Notes: Here, instead of moving the ball's position linearly, we are moving up and down according to a sine wave.
Challenge: Try changing the movement from up&down to side to side.
